// Тест кейсы для UI из итерации 2:

   Итерация 2:
   1. Депозит денег пользователем
   2. Перевод денег с одного аккаунта на другой
   3. Изменение имени пользователя

   Требования:
   1) Имя профиля - два слова сотсоящие только из букв обязательно
   2) Максимальная сумма депозита 5000
   3) Максимальная сумма перевода 10000

   Тест-кейсы для операций по депозиту:
   1) Пользователь может совершать депозит с валидными данными (позитивный)
   2) Пользователь не может делать депозит с невалидной суммой (негативный)

   Тест-кейсы для операций по трансферу:
   1) Пользователь может переводить деньги с одного счета на другой (позитивный)
   2) Пользователь не может переводить отрицательные суммы (негативный)
   3) Пользователь может отслеживать состояние своих учетных записей (позитивный)

   Тест-кейсы по изменении имени пользователем:
   1) Пользователь может меня свое имя в профиле (позитивный)
   2) Пользователь не может меня свое имя в профиле с некорректными данными (негативный)
